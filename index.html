Lightbox.js


A simple and easily customizable lightbox.


Download Get started

Get Started


Everything you'll need is located in the dist/ directory: include the javascript and css files in your HTML document:


<link href="dist/lightbox.css" rel="stylesheet">
<script src="dist/lightbox.js"></script>
